# GymManager

Sistema de gerenciamento de academias desenvolvido com ASP.NET Core e Clean Architecture.

## Tecnologias

- .NET 9
- ASP.NET Core
- Entity Framework Core
- SQL Server
- xUnit
- Swagger
- JWT

## Arquitetura

- Domain
- Application
- Persistence
- Infrastructure
- Api

## Funcionalidades

- Cadastro de alunos
- Matrículas
- Controle de presença
- Avaliação física
- Treinos
- Pagamentos
- Personal Trainer
- Cálculo de macronutrientes
- Controle de consumo de água
